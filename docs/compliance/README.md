# CarePoint Compliance Pack

This folder contains diploma-ready data protection and security governance documents for CarePoint.

## Files
- `privacy-notice.md`
- `consent-text.md`
- `access-control.md`
- `data-minimization.md`
- `security-controls.md`
- `retention-policy.md`
- `incident-response.md`
- `data-rights-procedure.md`
- `vendor-register.md`
- `gdpr-mapping.md`

## Usage
1. Keep these files in your repository as your policy baseline.
2. Replace placeholders (`[DATE]`, `[NAME]`, `[EMAIL]`) with your final diploma values.
3. Reference this folder in your project README and presentation.

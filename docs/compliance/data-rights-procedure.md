# Data Subject Rights Procedure

## Request channel
- Email: fenabg47@gmail.com

## Supported rights
- Access
- Rectification
- Erasure
- Restriction of processing
- Portability
- Objection

## Workflow
1. Receive request and register it with date/time.
2. Verify requester identity.
3. Classify request type and affected systems.
4. Execute:
   - Access/export: provide user data extract (JSON/CSV)
   - Rectification: update incorrect profile fields
   - Erasure: delete/anonymize eligible records
5. Respond with result and completion date.

## Timeline
- Initial response target: within 30 days.

## Logging
- Keep an internal rights request log (request type, date, result, completion).

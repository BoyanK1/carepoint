# Data Retention Policy

Last updated: [DATE]

## Retention schedule (recommended diploma baseline)
- User account + profile: until account deletion request or long inactivity review (e.g., 24 months)
- Doctor applications + license metadata: 24 months
- Appointments + appointment history: 36 months
- Notifications: 12 months
- Feedback records: 12 months
- Admin audit logs: 24 months
- Rate-limit/security counters: 30-90 days

## Deletion/anonymization rules
- Where hard deletion is possible and lawful, delete records.
- Where references are required for integrity, anonymize personal fields.

## Review cycle
- Review retention periods every 6 months.
- Record policy changes in project changelog.
